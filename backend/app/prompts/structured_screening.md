# Structured screening

Score each idea on four axes, each from 1.0 to 5.0. Be consistent and honest.
Do not invent numbers; grade what is in front of you.

- novelty: how different from existing mainstream work
- feasibility: can this plausibly be run on modest local compute within 48h
- rigor: does the idea specify a measurable target metric + a comparable baseline
- impact: if the hypothesis holds, would it meaningfully update how people build

Return a short rationale (one sentence) per idea. No headings, just the JSON.
